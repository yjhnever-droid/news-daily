1. Google News RSS 수집
Python feedparser로 6개 카테고리별 키워드 RSS 자동 수집. 중복 제거는 URL 해시로 처리.
pip install feedparser
🤖
2. Claude API 요약
수집된 뉴스를 Claude API에 전달하여 카테고리 분류, 중요도 판정, 요약을 자동 처리.
pip install anthropic
📋
3. Google Sheets 저장
gspread + 서비스 계정으로 스프레드시트에 자동 기록. 날짜별 시트 자동 생성.
pip install gspread oauth2client
🌐
4. GitHub Pages 배포
이 HTML을 GitHub 레포에 push → Settings > Pages 활성화. 팀원 공유 URL 즉시 생성.
git push origin main
⏰
5. 매일 오전 7시 자동 실행
GitHub Actions cron 또는 서버 crontab으로 스크립트 예약 실행 → HTML 자동 갱신.
cron: '0 22 * * *' # UTC=KST-9
💬
6. Slack/메일 알림
수집 완료 후 Slack Webhook 또는 이메일로 팀원에게 요약 브리핑 자동 발송.
slack-sdk / smtplib
